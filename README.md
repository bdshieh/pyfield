# pyfield
-----------

## Installation

### Create python environment
``` sh
conda create -n myenv python=3.6
conda activate myenv
```

### Install from local repository

``` sh
pip install .
```


### Install from remote repository
``` sh
pip install git+https://github.com/bdshieh/pyfield.git
```
